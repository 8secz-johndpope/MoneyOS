# ai manager web app

ai manager engine / live social innovation protocol

## ai world engine install

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your tests

```
npm run test
```

### Lints and fixes files

```
npm run lint
```

### Customize ai engine

Edit src/ai-config.json
